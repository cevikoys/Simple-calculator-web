# JavaScript Core Fundamentals & Interactive Web

Repositori ini berisi dokumentasi praktis dan proyek interaktif untuk memperkuat pemahaman fundamental **Vanilla JavaScript (Client-Side)**. Proyek di dalamnya mencakup manipulasi elemen halaman secara dinamis (DOM), arsitektur logika dasar, hingga studi kasus aplikasi kalkulator.

## Teknologi & Cakupan Keahlian
- **Teknologi:** HTML5, CSS3, JavaScript (Vanilla JS)
- **Kompetensi Inti:** Manipulasi DOM, Penanganan Event (`onclick`), Struktur Data (Object & Array), Logika Kontrol (Switch-Case & Perulangan), Pemrosesan Input Pengguna.

---

## Struktur Portofolio & Studi Kasus

### 1. Aplikasi Interaktif & Studi Kasus
* **Kalkulator JavaScript Sederhana (`kalkulatorJS.html`):**
  Aplikasi kalkulator aritmatika dinamis (Operasi Tambah & Kurang) yang memproses input pengguna secara *real-time* lewat manipulasi DOM, dilengkapi dengan kustomisasi tampilan visual (`css/style.css`).

### 2. Laboratorium Fundamental JavaScript (`latihan1.html` - `latihan10.html`)
Kumpulan berkas latihan terstruktur yang menunjukkan pemahaman mendalam tentang arsitektur JavaScript:
* **Integrasi Skrip:** Teknik penulisan kode JS internal maupun eksternal (`belajar.js`).
* **Manajemen Variabel & Tipe Data:** Implementasi tipe data *Number*, *String*, *Array*, dan *Object*.
* **Operasi Aritmatika:** Logika perhitungan matematika dasar dan operasi modulus.
* **Logika Kontrol & Fungsi:** Penerapan fungsi (*reusable function*), percabangan (`switch-case`), dan perulangan (`for-loop`) untuk menghasilkan konten dinamis.

---

## Cara Menjalankan Aplikasi
Seluruh proyek dalam repositori ini berjalan sepenuhnya di sisi klien (*client-side*). Anda tidak memerlukan server lokal (seperti XAMPP).
1. Clone repositori ini ke komputer Anda.
2. Buka berkas HTML apa pun (misalnya `kalkulatorJS.html` atau `latihan5.html`) secara langsung menggunakan web browser modern (Chrome, Edge, atau Firefox).
